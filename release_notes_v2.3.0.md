# NovelForge v2.3.0 更新日志

## 新增功能

- **卡片导入**：支持从其他项目导入角色、世界观、术语卡片
- **自动化设置**：新增日记自动添加日期、天气自动填充、章节自动编号、模板自动填充等开关
- **预设模板自定义**：角色预设模板（主角/配角/反派/路人）支持用户自定义编辑
- **主页主题切换**：启动器页面新增主题切换按钮
- **对话体角色插入**：对话体项目支持快速插入角色名称（类似剧本模式）

## 界面改进

- **右侧边栏扩展**：除时间线外，所有分类（角色/世界观/术语/分卷）均显示右侧文件列表
- **项目卡片优化**：书名添加《》书名号，修复背景模糊导致的文字不可读问题
- **类型选择面板**：抽屉高度增加，箭头改为">"并带旋转动画，创建按钮支持切换展开
- **项目名称书名号**：侧边栏和卡片中的项目名称统一添加《》
- **更多动画效果**：新增 fade-in、slide-up、scale-in、shake 等动画

## 功能修复

- **诗歌/歌词格式**：修复工具栏按钮无效的问题，现在可以正确切换诗歌居中排版和歌词缩进排版
- **共享世界章节创建**：修复同世界观系列项目的章节创建到正确子目录
- **项目存放目录**：扫描目录改为"项目存放目录"，支持自动创建不存在的目录
- **删除知识图谱入口**：移除侧边栏知识图谱入口（功能保留）

## 设置新增项

- 日记自动添加日期（默认开启）
- 天气自动填充（默认关闭）
- 章节自动编号（默认开启）
- 模板自动填充（默认开启）
